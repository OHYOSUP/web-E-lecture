네트워크 주요단어

get

값이나 속성을 가져오는(데이터를 받아오는) 통상적인 단어로 많이 사용한다.

set

값이나 속성을 만들어내는(데이터자체를 생성하는) 통상적인 단어로 많이 사용한다.

require

요청한다는 뜻으로

  1. 브라우저에서 웹 서핑을 하는 행위를 세분화 할 수 있다.
  2. 클라이언트(사용자)의 시스템에 설치된 브라우저를 조작하여 주소를 적는다.
  3. 주소 = URL,domain,ip 등등 여러가지 경로를 말한다.
  4. 주소에 접속했을때 서버에게 웹문서의 데이터를 받아오는 과정을 거친다.
  5. 데이터를 '요청(require)' 한다고 부른다. 사실상 웹페이지의 데이터를 다운로드 하는 과정이다.

reponse

응답한다는 뜻으로

  1. 서버(server)가 사용자(client)에게 데이터를 전송해주는 과정을 뜻한다.
  2. 요청이 오면 응답하는 일련의 네트워킹 시스템이 마련되어 있고, 이것을 '통신' 이라고 부른다.
