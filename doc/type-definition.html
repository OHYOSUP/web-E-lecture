type(타입)

ECMAscript 명세에 따르면,
'타입'이란 자바스크립트 엔진 개발자 모두에게 어떤 값을 다른 값과 분별할 수 있는, 고유한 내부 특성의 집합이다.

    built-in type(내장타입)

  1. null
  2. undefined
  3. boolean
  4. number
  5. stiring
  6. object
  7. symbol(ES6 버전부터)

null

mdn 명세

어떠한 값 자리가 존재는 하나 '값이 비어있음' 을 뜻한다. 정의되지 않는 undefined와 개념을 구분한다.

undefined

mdn 명세

단어 그대로 '정의되어있지 않다.' 라는 뜻이며, 일반적으로 값이 할당되지 않을때 해당 타입을 확인 할 수 있다.

boolean

number

정수(integer) : 소수점이 없는 숫자 / 양의 정수, 0, 음의 정수

소수(prime number) : 0.1 등 소수점 범위까지 (

주요 메서드

parseInt("문자열인 숫자") -> 정수로된 숫자타입으로 형변환

parseFloat("문자열인 숫자 실수") -> 소숫점 까지의 범위의 실수를 숫자타입으로 형변환

parse : 파스, parsing : 파싱

특정 값을 분석하기 위한 방식으로 종종 데이터타입을 '분석' 해서 사용할 때가 있다.

ex> prompt 라는 메서드를 사용하면 사용자에게 입력을 받을 수 있는데, 사용자가 전달해주는 모든 입력데이터는 "문자열" 이다. 받은데이터가 숫자여야 할때 parseInt() 메서드를 사용하는 것을 자주 발견 할 수 있다.
